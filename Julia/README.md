This is the Julia implementation of Conway's game of life. 

To use it, include the file in your Julia REPL and call the function life(grid, steps) 

The currently included patterns are :
* Block
* Beehive
* Loaf
* Boat 
* Tub
* Blinker 
* Toad
* Beacon
* Pulsar 

Future additions :

* Infinite grid
